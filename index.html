The boxes in the box model

The Content Box

The inner core of the box model is the content (otherwise it is an empty box).

Properties that can be set on the content box are:

  • height
  • width
  • max-height
  • min-height
  • max-width
  • min-width

Note: box heights and borders do not observe % values for length

The Padding Box

Padding refers to the inner margin of a CSS box
(between the outer edge of content box and inner edge of border)

Padding can be set in a number of ways:

  • padding: 1rem;Sets 1rem of padding on each side
  • padding: 1rem 2rem;Sets 1rem of padding on the top and bottom and 2rem of padding on each side
  • padding: 1rem 3rem 2rem;Sets 1rem of padding on the top, 3rem of padding on the right and left, and 2rem of padding on the bottom
  • padding: 1rem 1.5rem 2rem 1rem;Sets 1rem of padding on the top, 1.5rem of padding on the rigth, 2rem of padding on the bottom and 1rem on the left (clockwise starting at 12 o'clock)

The Border Box

The border sits between the outer edge of the padding and inner edge of the margin.

The border shorthand property sets the values of border-width, border-style, and border-color.

It looks like this: border: solid 2px black;

Border Styles

  • No Border
  • Hidden
  • Dotted
  • Dashed
  • Solid
  • Double
  • Groove
  • Ridge
  • Inset
  • Outset

Border Radius

The corners are right angles but can also be rounded using border-radius (box-shadow will follow suit).

  • No Radius
  • Round Radius
  • Rounder Radius
  • Roundest Radius

Differing radius can be set for each corner too...

  • Top Right and Bottom Left vs Top Left and Bottom Right
  • Top vs Bottom
  • Right vs left
  • Just the corner
  • Wonky

And another thing about borders...

  • border-radius: 50% 50% 0 0 / 100% 100% 0 0;

Border longhand

Another way of doing it is instead of using the border shorthand you can set different styles (or size or colour)for each side:

  • border-style: dotted;All four sides are dotted
  • border-style: dotted solid;Top and Bottom are dotted, right and left are solid
  • border-style: hidden double dashed;Top is hidden, right and left are double and bottom is dashed
  • border-style: none solid dotted dashed;Top is none, right is solid, left is dotted and bottom is dashed

The Margin Box

The margin surrounds a CSS box and pushes up against other CSS boxes in the layout..it is styled very similarly to padding

Margins can be set in a number of ways:

  • margin: 0;No margin
  • margin: none;Also no margin
  • margin: 1rem;Sets 1rem margin on each side
  • margin: 1rem 2rem;Sets 1rem of margin for the top and bottom and 2rem of margin for each side
  • margin: 1rem 3rem 2rem;Sets 1rem of margin for the top, 3rem for the margin for the right and left, and 2rem of margin for the bottom
  • margin: 1rem 1.5rem 2rem 1rem;Sets 1rem of margin for the top, 1.5rem of margin for the right, 2rem of margin for the bottom and 1rem on the left (clockwise starting at 12 o'clock just like padding)

Sizing up your Box

Box heights and borders don't observe percentage lengths

Vertical margins collapse so only the margin with the largest value is used (instead of both values added together)

Overflowing your Box

When the size of a box is set with absolute values the content may not fit within the allowed size and will overflow

The overflow property has some common values:

  • autoThe overflowing content is hidden and scroll bars are shown
  • hiddenThe overflowing content is hidden
  • visibleThe overflowing content is shown outside the box (default)

You can also specify overflow-x and overflow-y

Background clip

Box backgrounds are made up of colours and images stacked on each other (background-color, background-image)

By default extends to the outer edge of the border (can be adjusted using background-clip property on the box)

The background-clip property sets whether an element's background extends underneath its border box, padding box, or content box.

  • Control - no clip specified
  • Border
  • Padding
  • Content
  • Text
