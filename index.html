Thinking inside the box (model)!

The boxes in the box model

Note: box heights and borders do not observe % values for length

Overflow

When the size of a box is set with absolute values the content may not fit within the allowed size and will overflow

The overflow property has some common values:

  • autoThe overflowing content is hidden and scroll bars are shown
  • hiddenThe overflowing content is hidden
  • visibleThe overflowing content is shown outside the box (default)

Background clip

Box backgrounds are made up of colours and images stacked on each other (background-color, background-image)

By default extends to the outer edge of the border (can be adjusted using background-clip property on the box)

Outline

Looks like a border but is not part of the box model and is drawn on top of the box without changing the size of the box (drawn outside the border box inside the margin area)

Outline is used for accessibility reasons to highlight an active part of the page such as links – don’t make your outlines look like link highlights.
